

# 1. 语法 Hello Markdown ！

## 1.1 标题
    在标题前添加前缀#，例如：
    # 一级标题
    ## 二级标题
    ### 三级标题

## 1.2 段落
    方法一：行末添加两个以上的空格，加回车
    方法二：段落后面添加一个空行

## 1.3 字体
  *斜体*,  **粗体**,  ***粗斜体*** :

    *斜体*
    _斜体_

    **粗体**
    __粗体__

    ***粗斜体***
    ___粗斜体___

# 2. 工具 + 快捷键

  " markdown-preview,
